Задачи: Разработка лендинга на неделю (17.11 - 21.11)

Обзор недели

На этой неделе наша главная цель — полностью разобраться с функционалом Tilda, включая тарифы, возможности, работу с шаблонами и блоками. Затем, используя полученные знания, собрать современный, адаптивный лендинг, готовый к публикации и продвижению, с настроенным доменом и базовыми интеграциями.

Главная цель

Note

Полностью освоить платформу Tilda и создать современный лендинг, готовый к публикации и базовому SEO-продвижению, с подключением домена.

Вторник (18.11)

  • Практическое применение: Начало сборки лендинга (4 часа)
    • 1. Выбор концепции и создание нового проекта (1 час)
      • Цель задания: Выбрать общую концепцию лендинга, опираясь на вдохновение из Figma, и создать новый проект в Tilda.
      • Чему научимся в конце: Определять стиль и структуру будущего лендинга, а также уверенно инициировать новый проект на платформе.
      • Справочный центр Tilda:
        • Создание новой страницы: [https://help-ru.tilda.cc/page-actions#rec15520104] [1]
        • (Рекомендуется также пересмотреть "Знакомство с Тильдой" (видео 2) и "Создание сайта из базовых блоков" (видео 5)).
      • Рекомендуется для вдохновения (макеты Figma):
        • Простые шаблоны сайтов: https://rejump.ru/20-maketov-sajtov-sdelannyh-v-figma-vse-stranitsy-s-adaptivom]
    • 2. Сборка шапки сайта (Header) (1 час)
      • Цель задания: Разработать навигационную часть лендинга, включающую логотип, меню и, при необходимости, контактные данные, используя стандартные блоки Tilda.
      • Чему научимся в конце: Уверенно использовать готовые блоки для Header, настраивать их внешний вид, добавлять пункты меню и обеспечивать удобство навигации на сайте.
      • Справочный центр:
        • Шапка и подвал: [https://help-ru.tilda.cc/header-footer] [1]
        • Как сделать меню: [https://help-ru.tilda.cc/menu] [1]
    • 3. Сборка первого экрана (Hero-секции / Обложки) (1 час)
      • Цель задания: Создать привлекательный и информативный первый экран лендинга, который захватывает внимание пользователя и передает ключевое сообщение.
      • Чему научимся в конце: Эффективно использовать блоки "Обложка" Tilda для размещения заголовка, подзаголовка, кнопки призыва к действию (CTA) и фонового изображения, а также применять базовые настройки адаптивности для этого блока.
      • Справочный центр:
        • Работа с обложкой: [https://help-ru.tilda.cc/cover] [1]
        • (Также можно кратко пересмотреть "Настройка адаптивности" (видео 7) для лучшего понимания отображения на разных устройствах).
    • 4. Базовая настройка шрифтов и внутренняя навигация (1 час)
      • Цель задания: Применить общие настройки шрифтов для соответствия стилю лендинга и настроить базовые якорные ссылки для навигации внутри страницы.
      • Чему научимся в конце: Изменять шрифты для улучшения визуальной читаемости, применять их ко всему сайту и создавать якорные ссылки для плавных переходов к разделам на одностраничном лендинге.
      • Справочный центр:
        • Настройка шрифтов: [https://help-ru.tilda.cc/fonts] [1]
        • Якорные ссылки: [https://help-ru.tilda.cc/anchor] [1]
    • Общее время на практические задания Вторника: 4 часа

Среда (19.11)

  • Сборка информационных секций: Создание 2-3 основных контентных блоков (например, "О нас", "Преимущества", "Как это работает") с использованием различных блоков Tilda.
    • Время: 5 часов
  • Работа с медиаконтентом: Загрузка и оптимизация изображений, видео для сайта в рамках Tilda.
    • Время: 3 часа

Четверг (20.11)

  • Сборка секций с интерактивными элементами: Реализация формы обратной связи/подписки, секции с отзывами/галереями с использованием встроенных блоков Tilda.
    • Время: 4 часа
  • Верстка Footer: Создание нижней части сайта с контактной информацией, ссылками, копирайтом.
    • Время: 2 часа
  • Адаптивная настройка (мобильная/планшетная версии): Обеспечение корректного отображения всех элементов лендинга на различных устройствах с помощью встроенных инструментов Tilda (особенно для Zero-блоков).
    • Время: 2 часа

Пятница (21.11)

  • Базовая SEO-оптимизация на Tilda: Настройка мета-тегов (Title, Description), alt-атрибутов для изображений, заголовков H1-H6 с использованием настроек страниц и блоков Tilda. 1
    • Время: 3 часа
  • Подключение домена: Настройка собственного домена для опубликованного лендинга в Tilda. 1, 1
    • Время: 2 часа
  • Интеграции: Подключение форм к CRM или почте, установка счетчиков аналитики (Google Analytics/Яндекс.Метрика) через настройки Tilda.
    • Время: 2 часа
  • Тестирование и подготовка к ревью: Проверка работы всех форм, ссылок, адаптивности на разных устройствах. Финальная проверка перед демонстрацией.
    • Время: 1 час

Коммуникация

  • Ежедневные короткие стендапы (15 минут) для обсуждения прогресса и возникающих вопросов.
  • Оперативная связь в мессенджере по мере необходимости.
